# Инструменты (Tools)

Этот документ содержит описание доступных инструментов, используемых в коде проекта (`ui/model/update.go`), их аргументов и форматов обработки данных.

## 1. Write
Используется для записи или создания файлов.

### Аргументы
| Поле | Тип | Описание |
| :--- | :--- | :--- |
| `file_path` | string | Путь к файлу. |
| `content` | string | Содержимое для записи. |

### Обработка результатов
- Если результат содержит `Diff`, он рендерится через `messages.RenderDiff`.
- Если содержит `type: create`, возвращается "File created".
- В остальных случаях возвращается "File updated".

---

## 2. Read
Используется для чтения файлов.

### Аргументы
| Поле | Тип | Описание |
| :--- | :--- | :--- |
| `file_path` | string | Путь к файлу. |
| `offset` | float64 | Номер строки начала чтения (необязательно). |
| `limit` | float64 | Количество строк для чтения (необязательно). |

### Обработка результатов
- Возвращает количество прочитанных строк и размер в байтах в формате: `Read N lines (X B/KB)`.

---

## 3. Glob
Используется для поиска файлов по шаблону.

### Аргументы
| Поле | Тип | Описание |
| :--- | :--- | :--- |
| `pattern` | string | Паттерн поиска. |
| `path` | string | Директория поиска. |

### Обработка результатов
- Если первая строка содержит "Found" или "No", возвращается она же.
- Иначе возвращается количество результатов: `N results`.

---

## 4. Grep
Используется для поиска текста в файлах.

### Аргументы
| Поле | Тип | Описание |
| :--- | :--- | :--- |
| `pattern` | string | Текст для поиска. |
| `path` | string | Директория поиска. |
| `glob` | string | Фильтр файлов. |
| `output_mode` | string | Режим вывода (`content`, `count` или пустой). |

### Обработка результатов
- Возвращается форматированная строка с описанием условий поиска и режим вывода, либо количество найденных результатов.

---

## 5. Edit
Используется для редактирования файлов.

### Аргументы
| Поле | Тип | Описание |
| :--- | :--- | :--- |
| `file_path` | string | Путь к файлу. |
| `diff` | string | Дифф изменений. |
| `replaceAll` | bool | Флаг замены всех вхождений. |

### Обработка результатов
- Если результат содержит `Diff`, рендерится через `messages.RenderDiff`.
- В остальных случаях возвращается "File edited".
